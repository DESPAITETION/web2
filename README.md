# Drupal-coder Website

Современный адаптивный сайт для Drupal-coder, разработанный на React + Vite.

## 🚀 Деплой на GitHub Pages

Сайт доступен по адресу: https://despaitetion.github.io/web/

## 📦 Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Деплой на GitHub Pages
npm run deploy
```

## 🎨 Особенности

- ✅ React 18 + Vite
- ✅ Mobile-first responsive design
- ✅ Интерактивное меню с анимациями
- ✅ Форма обратной связи с валидацией
